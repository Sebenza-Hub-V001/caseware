# CaseWare Probe Project Overview

## Summary

A comprehensive briefing on the CaseWare Probe How-To Document Project — a self-initiated effort to build a library of professional training guides for the [[entities/probe-audit-premium-plus]] methodology used within [[entities/caseware-working-papers]]. The project is structured around a [[concepts/three-stage-pipeline]] (V1 How-To → V2 Enhanced → Video Script) and aligned to [[concepts/isa-standards]].

## Key Claims

- Each working paper follows a **three-stage production pipeline**: base How-To (V1), enhanced with ISA-aligned guidance tables (V2), and narrated video script (V3).
- Documents are produced as `.docx` files using a standardised [[concepts/ooxml-document-structure]] (Montserrat headings, custom colour palette, numbered sections, branded header/footer).
- The V2 enhancement cross-references working papers against the **Probe Audit Manual**, identifying 3–5 critical questions per paper with ISA linkage.
- Video scripts include production cues, section transitions, and a Video Details metadata table.
- Working papers are tracked by CaseWare reference numbers (e.g., 02.40 Evaluation of Misstatements, 12.50).
- The project serves dual purposes: personal learning tool and potential resource for other audit clerks.
