# CaseWare Working Papers

CaseWare Working Papers is the audit software platform used to manage and complete engagement working papers. It includes **CaseView**, the document viewer/editor for working paper templates.

## Role in the Project

- All How-To documents are based on CaseView PDF exports of the original working paper templates.
- Video scripts are designed as screen-based walkthroughs recorded within CaseWare.
- Working papers are identified by reference numbers (e.g., 02.40, 12.50).

## Key Features

- Working paper templates with structured fields and questions
- CaseView for document viewing and completion
- Reference numbering system for working papers
